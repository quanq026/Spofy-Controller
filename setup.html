Spotify App Credentials

Tạo ứng dụng trên Spotify Developer Dashboard để lấy Client ID và Client Secret.

Hướng dẫn:
1. Truy cập developer.spotify.com/dashboard
2. Tạo App mới → Copy Client ID và Client Secret

GitHub Gist (Token Storage)

Gist được sử dụng để lưu trữ Spotify tokens một cách an toàn.

Hướng dẫn:
1. Tạo Gist mới (private/public)
2. Copy Gist ID từ URL (phần sau gist.github.com/username/)
3. Tạo Personal Access Token với quyền "gist"

Lấy từ URL: gist.github.com/username/[GIST_ID]

Token cần có quyền "gist" để đọc/ghi
