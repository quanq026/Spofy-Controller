Self-hosted Spotify Remote

Điều khiển Spotify
từ bất kỳ đâu

Ứng dụng web cho phép bạn điều khiển Spotify trên mọi thiết bị. Tự host, bảo mật, và hoàn toàn miễn phí.

Tạo tài khoản Đã có tài khoản

Multi-user

Mỗi người dùng có tài khoản riêng với Spotify credentials và Gist storage độc lập. Dễ dàng chia sẻ server cho nhiều người.

Bảo mật

Token được mã hóa và lưu trên GitHub Gist của bạn. Session riêng biệt cho mỗi user, hỗ trợ auto-refresh token.

REST API

Hỗ trợ API Key để tích hợp với các ứng dụng, scripts, hoặc tự động hóa workflow của bạn.

Player trực quan

Giao diện hiện đại, dễ sử dụng với các tính năng cơ bản và được cập nhật liên tục.

Multi-device

Điều khiển từ máy tính, điện thoại, hoặc tablet. Responsive design hoạt động mượt trên mọi màn hình.

Dễ cài đặt

Setup wizard hướng dẫn từng bước. Chỉ cần Spotify App + GitHub Gist, không cần server database phức tạp.
